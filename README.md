# UA Esports

UA Esports is a real-time esports contest platform that allows players to participate in competitive matches across multiple games. Users can join contests, view match details, track match status, check results, and manage their profiles through a modern and user-friendly interface.

## Features

* User Authentication (Login & Registration)
* Real-Time Contest Updates
* Upcoming, Ongoing, and Completed Matches
* Dynamic Match Cards
* Multi-Game Support
* User Profile Management
* Firebase Firestore Integration
* Responsive and Modern UI
* Real-Time Data Synchronization

## Tech Stack

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* Firebase Storage

## Getting Started

1. Clone the repository.
2. Install dependencies:

flutter pub get

3. Configure Firebase:

flutterfire configure

4. Run the application:

flutter run
